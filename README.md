# 🫒 Olive App Clone

A pixel-perfect recreation of [oliveapp.com](https://www.oliveapp.com/) built as a frontend assessment for Hire My Idea (Praxso).

**Built by: Om Gupta**

---

## 🚀 Tech Stack

- **React 19** + **TypeScript** + **Vite**
- **Framer Motion** — smooth animations, carousel transitions, floating effects
- **Lucide React** — icons
- **Tailwind CSS** — utility styling
- **Unsplash** — high quality images

## ✨ Features Implemented

- ✅ Sticky navbar with scroll effect
- ✅ Hero section with floating phone mockup animation
- ✅ Stats counter section (50K+ families, 1M+ scans)
- ✅ How it Works — 3 cards with hover lift effect
- ✅ Features — Nutritional Clarity, Ingredient Filtering, Health Outcomes
- ✅ Testimonials with auto-sliding Unsplash images
- ✅ "Protect Your Family" animated image carousel
- ✅ Comparison table (Olive vs competitors)
- ✅ Pricing cards (Monthly / Yearly)
- ✅ 100% Independent section
- ✅ FAQ accordion
- ✅ CTA Banner with real grocery image
- ✅ Footer with newsletter signup
- ✅ Scroll progress indicator
- ✅ Fully responsive

## 🛠️ Run Locally

```bash
npm install
npm run dev
```

## 📦 Build

```bash
npm run build
```
